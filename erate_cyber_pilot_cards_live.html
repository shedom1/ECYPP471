E-Rate Cybersecurity Pilot Program Live Viewer

Grouped service-provider view for the USAC Cybersecurity Pilot FCC Form 471 dataset. Cards are bundled by Status + Participant Name + Provider + State + Application + Revised FCDL Date. Each FRN gets its own line-item table so the make, model, description, quantities, and amount fields are easier to read like a PO or invoice.

This version follows the glossary field structure for Participant Name, FRN, FRN Line Item Number, make/model, monthly and one-time line-item amount fields, while keeping FRN Requested Amount and FRN Current Committed Amount as display-only FRN values instead of summing them across rows.
